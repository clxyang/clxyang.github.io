About me

Featured Projects

Here are some projects I've completed so far. I'm excited to add more to the list:

Disease Categorization in Text

Trained a Convolutional Neural Network model for disease categorization based on consumer posts on Reddit. Achieved 95%+ accuracy in predicting the "silver standard", being the name of the subreddit of the post. Saw a 14.33% increase in F1 score compared to a traditional Naive Bayes classifier.

Auditory Hallucinations on Reddit

Gathered and analyzed 40,645 Reddit posts using Python scripts. Studied patterns in auditory hallucinations within and across different mental health diagnostic categories using semantic analysis and topic modeling.

BillCitizen

Created a mobile app using React Native that decodes the legal jargon in legislative documents, allows individuals to track local legislation in an intuitive manner, and makes it quick and easy to take tangible action. Developed the app using Node.js and TypeScript.

SoundBio iGEM Bacto-Basics

Co-led the SoundBio iGEM team in executing our project Bacto-Basics, which aimed to functionalize bacterial cellulose through the attachment of chromoproteins to the cellular membrane. Created a platform for precise, light-based control of this functionalization for applications including burn wound treatment and tissue scaffolding.

Other Noteworthy Work

Travel Planner

Implemented Dijkstra's algorithm to compute the most optimal route between locations on a map, prioritizing either price, time, or directness.

Search

Created a search engine able to index XML documents and process search queries using Google's PageRank algorithm. The search engine calculates and outputs the top 10 most relevant documents for any free text query.

Updater

Designed a "cursor" data structure that represents an immutable tree of arbitrary size, allowing for much more efficient tree traversal and modification.

Seamcarve

Used dynamic programming and memoization to resize images by removing the least important vertical "seams", determined by the difference in color between each pixel and its neighbors.

Decision Tree

Wrote a program able to make accurate classifications of new items based on data read in from a dataset. Used a tree-shaped data structure, a "decision tree", to model attributes and outcomes.
